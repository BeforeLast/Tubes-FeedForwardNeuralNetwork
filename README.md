# Tubes-FeedForwardNeuralNetwork

## Description
Implementation of Feed Forward Neural Network.

## Features
- Read Model
- Visualize Model
- Single Prediction
- Batch Prediction

## Requirements
#### Libraries
- graphviz v0.19.1
- ipython v8.1.0
- pdf2image v1.16.0

#### Program
- graphviz

#### Install requirements
1. Run "pip install -r requirements.txt" at terminal
2. Install graphviz by downloading from [Graphviz]
